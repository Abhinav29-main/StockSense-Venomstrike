Claw & Shield Track
🦞 OpenClaw Agent Interface

StockSense runs as an OpenClaw skill — your personal AI agent can analyze stocks directly through WhatsApp, Discord, Telegram, or Slack. Every tool call is protected by ArmorIQ: intent tokens, prompt-injection guards, and a fail-closed policy engine.

🦞 OpenClaw + StockSense skill
🛡️ ArmorIQ — unverified
Simulates: OpenClaw agent → /api/openclaw/{TICKER}
Try an agent command:
Analyze AAPL Should I buy NVDA? Signal on TSLA? MSFT bullish/bearish? Run StockSense on AMD
You (via WhatsApp/Discord) > waiting for command… // OpenClaw receives your message, extracts the ticker, // calls GET /api/openclaw/{TICKER} with ArmorIQ intent verification.
🛡️ ArmorIQ Enforcement Log
01 · OPENCLAW SKILL
StockSense ships a skills/stocksense/SKILL.md file. OpenClaw reads it, learns the endpoint, and auto-dispatches stock queries from any connected channel.
02 · ARMORIQ SHIELD
Every agent tool call generates an intent token. ArmorIQ verifies the token and the approved plan before execution — blocking prompt injection and data exfiltration attempts.
03 · FAIL-CLOSED POLICY
ArmorIQ's policy allows only web_fetch(localhost:5050). Any attempt to access external URLs, read system files, or run shell commands is blocked before execution.
